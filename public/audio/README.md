## How to Add Your Audio Files

1. Go to your GitHub repository
2. Navigate to `/public/audio/`
3. Upload your 16 watermarked preview files (drag and drop or use the "Add file" button)
4. Commit the changes
5. Netlify will automatically rebuild and deploy your site

## Important Notes

- All files should be in MP3 format for best browser compatibility
- Keep file sizes reasonable (30-second previews should be small)
- Make sure each file has your watermark to protect your work
- File names must match exactly (including hyphens and lowercase)
